Portfolio Website Case Study

This project details the design and development process of this very portfolio website, showcasing my skills as an Interaction Designer.

Timeline: ∞, infinite/none, updated iteratively Role: UX Designer + Programming Platform: Website Programming languages: HTML, Tailwind CSS, Javascript Tools: Netlify, GitHub

Project Goal

The primary goal was to create a clean, modern, and responsive website to effectively showcase my UX/UI design projects and skills. It needed to be easy to navigate and provide a clear overview of my work and capabilities.

Technologies Used

  • HTML5: For structuring the content.
  • Tailwind CSS: For rapid styling and responsive design.
  • JavaScript: For interactive elements like the sticky navigation and smooth scrolling.
  • Custom CSS Animation: For the seamless logo carousel effect.

Key Features Developed

  • Responsive Layout: Ensuring the site looks great on all devices.
  • Sticky Navigation: The header stays visible as you scroll.
  • Smooth Scrolling: Providing a pleasant user experience when navigating between sections.
  • Seamless Logo Carousel: A continuously animating display of tools and technologies used.

Design Process

The design process involved sketching, wireframing, and prototyping to define the layout and user flow. Emphasis was placed on clear typography, a balanced visual hierarchy, and a modern aesthetic using Tailwind CSS.

Reflection

Building this portfolio provided valuable experience in front-end development, integrating design principles with code. The focus on responsiveness and performance enhancements for elements like the carousel were key learning points.
